# BLS Pace Setter

A simple, open-source web-based tool to assist clients with bilateral stimulation (BLS) for EMDR therapy. This tool serves as a supplementary resource that clients can use between sessions. It is privacy-focused, client-side only, and customizable.

## Planned Features

- Visual bilateral stimulation with a moving dot
- Adjustable speed control
- Start/stop functionality
- Responsive design for various devices
- Accessibility considerations (keyboard navigation, screen reader support)

## How to Run Locally

1. Clone the repository:
   ```
   git clone https://github.com/z-teckson/bls-pace-setter.git
   ```
2. Open `index.html` in your web browser.
3. No server required - all code runs client-side.
