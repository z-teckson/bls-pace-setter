BLS Pace Setter

A privacy‑focused bilateral stimulation tool for EMDR therapy

How to Use

Click the Start button to begin the bilateral stimulation. The dot will move horizontally across the screen. Use the Stop button to pause the animation.

This tool is intended as a supplementary resource between therapy sessions. Always follow your therapist's guidance.

Visual Stimulator

About This Project

This is an open‑source tool developed by mental health professionals. The code is entirely client‑side – no data leaves your browser. You can inspect, modify, and contribute to the project on GitHub.

Made with by psychologists passionate about accessible mental health tools.
